DicoGIS (formerly DicoShapes)

EN:

Summary

Automatize the creation of a dictionnary of geographic data in a folders structure. The output dictionary is an Excel file (.xls).

History

Developped in Python 2.7.x for the PACIVUR research program which is part of the IRD (French Institute of Research for the Development). Compiled in executable with py2exe.

ES:

Resumen

Automatiza la creación de un diccionario de los datos geográficos (shapefiles y tablas MapInfo) contenidos en una estructura de carpetas. El archivo final es una tabla Excel 2033 (.xls).

Histórico

Desarrollado en Python 2.7.x para el programa de investigación PACIVUR (Programa Andino de Capacitación e Investigación sobre Vulnerabilidad y Riesgos en medio urbano) que es parte del IRD (instituto francés de investigación para el desarrollo). Compilado en ejecutable con py2exe.

FR :

Résumé

Crée un dictionnaire des données géographiques (shapefiles et tables MapInfo) contenues dans une arborescence de répertoire. Le dictionnaire prend la forme d'un fichier Excel 2003 (.xls).

Historique

Développé en Python 2.7.x pour le programme de recherche PACIVUR de l'IRD (Institut de Recherche pour le Développement). Compilé en exécutable avec py2exe.
